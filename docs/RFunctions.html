Slicing and extracting data

Indexing vectors

  1. x[n] nth element Calls out the nth element
  2. x[-n] all but nth element Calls out all but the nth element
  3. x[1:n] first n elements Calls out first n elements
  4. x[-(1:n)] elements from n+1 to the end Calls out elements from n+1 to the end
  5. x[c(1,4,2)] specific elements Calls out specific elements
  6. x[“name”] elements named “name” Calls out the name of some element
  7. x[x>3] all elements greater than 3 Calls out all elements greater than 3
  8. x[x > 3 & x < 5] all elements between 3 and 5 Calls out all elements between three and five
  9. x[x %in% c(“a”,“and”,“the”)] all elements in given set Calls out all elements of a given set

Indexing lists

  1. x[n] list with elements n
  2. x[[n]] nt h element of the list
  3. x[[“name”]] element of the list named “name”
  4. x$name id.

Indexing matrices

  1. x[i,j] element at row i, column j Calls out element at row i and column j
  2. x[i,] row i Calls out row i
  3. x[,j] column j Calls out column j
  4. x[,c(1,3)] columns 1 and 3 Calls out columns 1 and 3
  5. x[“name”,] row named “name” Calls now the row named “name”

Indexing data frames (matrix indexing plus the following)

  1. x[[“name”]] column named “name” Calls out the column named “name”
  2. x$nameid.
